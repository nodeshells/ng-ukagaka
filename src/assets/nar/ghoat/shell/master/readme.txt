今回は女の子？のFSを描いてみました。
前のFS『SHLEEP』を意識して描いたものですので、シリーズものになりそうです。一緒に使って頂いても嬉しいです！
アニメーション部分を頑張ってみましたが、結果的にこのサーフェス設定に慣れてない方には迷惑をかけるばかりではないかと、自覚しました。

【著作權についての声明】
理論上SSPに作動する伺か専用のFSです。
フリーシェルとは言え、著作權を放棄する訳ではありませんので二次配布、直リングと自作発言はやめてください。
お手数かもしれませんが、ご使用になる場合、もしお作品の配布先を教えて頂けれると嬉しいです。

【使用について】
シェルのエラー（surface.txtの文法ミス、シェルの透明な部分）を修正することは許可されています。
大抵シェルキャラの改変（カラー、服、髪型等の改変）は禁止されていますのでご注意ください。

初心者ではない方にとって退屈かもしれませんが、以下Surface.txtと表情パーツについて説明させていただきます。

【このFSの表情の組み合わせ方法説明】
デフォルト表情（０）を例に、
surface0はご覧の通りのっぺらぼうでございますが、実際は

element0,base,surface0.png,0,0で
element1,overlay,surface1001.png,0,0（眉）
element2,overlay,surface1010.png,0,0（目）
element3,overlay,surface1101.png,0,0（口）

の三部分を組み合わせて、初めて顔と呼ばれるものになります。

【表情説明】
アニメの説明はSurface.txtにありますのでここで述べません。
表情は１-９、１１-１３、３０-４４、５０-５４を用意しました。

１-９　　　普通の表情。悲しんだり笑ったり、表情テスト機能で簡単に見れます。トーク、瞬きアニメ付き。
１１-１３　普通の表情２。特殊口型の使用。悲しんだり笑ったり、表情テスト機能で簡単に見れます。トーク、瞬きアニメ付き。
３０-４４　赤面状態の表情。悲しんだり笑ったり、表情テスト機能で簡単に見れます。トーク、瞬きアニメ付き。

パーツの組み合わせによる表情の追加はご自由にどうぞ。
